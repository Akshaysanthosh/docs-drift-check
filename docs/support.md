# Support

## Best support path

- Bug reports: open a GitHub issue
- Feature requests: open a GitHub issue
- Security issues: do not open a public issue, use the contact path in [SECURITY.md](../SECURITY.md)

## Scope

This project currently supports:

- the repo-local skill workflow
- the local Codex plugin workflow
- the bundled validation and smoke-test scripts in this repository

## Response expectations

This is an independently maintained project. Response times are best-effort rather than guaranteed SLA support.

## Good bug reports

Include:

- the repository or repo shape you tested against
- the base branch used
- the exact prompt or command
- expected behavior
- actual behavior
- whether the issue was a false positive, false negative, or install problem
