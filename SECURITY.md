# Security Policy

If you believe you have found a security issue in Docs Drift Check, please do not open a public GitHub issue first.

## Reporting

Send a private report to:

- `sakshay797@gmail.com`

## What to include

- a clear description of the issue
- reproduction steps
- affected files or workflows
- expected impact

## Scope

Relevant reports include:

- unsafe file handling in bundled scripts
- unexpected data exposure in plugin behavior
- installation or packaging issues that create meaningful security risk

Best-effort acknowledgment and triage applies.
